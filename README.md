# Diamant Backend

A high-performance, lightweight x86-64 machine code generator and AST compiler written from scratch in Python using SLY. 

Unlike traditional compilers that rely on heavy infrastructure or third-party assemblers, **Diamant** tokenizes high-level algebraic expressions, constructs a recursive abstract syntax tree (AST), flattens the tree via a stateful walker, and emits raw, executable x86-64 machine code bytes (`bytearray`) directly.

## Features

* **Implicit Parsing Engine:** Tokenization configuration allows robust, spaces-optional syntax slicing (e.g., `rax=100rax=20rbx=2000` evaluates flawlessly).
* **Dynamic Expression Cascading:** Recursively flattens multi-register algebraic chains using a flexible binary tree model rather than a rigid, single-destination accumulator.
* **Pure Binary Emitter:** Handles architectural layout requirements under the hood—calculating REX prefixes, ModR/M byte mode fields, and size-override opcodes manually.
* **Separation of Concerns:** Distinct separation between math computation loops and final high-level scope variable assignments (`reg = expr`).
